# My portfolio

## Install

This repository contains the code for my portfolio, hosted on quentinaudinot.heroku.com

Installation :
```
npm install
bower install
gulp
npm start
```

## Technologies

This portfolio has been made using Angular.js on the front-end and Node.js, Express and MongoDB on the back-end.
